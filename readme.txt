BGMはSuno AIを使用 https://suno.com

ゲーム内容
2人の自機を操作し、より早くゴールを目指すランゲーム

操作方法
画面遷移→エンターキー

プレイヤー1（N）プレイヤー2（S）
移動→S,Dキー
ジャンプ→スペースキー
アクション→Mキー
操作キャラ変更ー＞Nキー
